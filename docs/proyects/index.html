Managing GitHub Users and Repositories

Github API | Vue | JavaScript | HTML | Bootstrap | Github

This lab aims to create a complex Vue application using single-file components. The application is a single-page application that will allow the user to visualize certain data available through the GitHub API.

  • Developed a Vue.js SPA using three reusable components (App, GitHub, GitHubRepo).

  • Implemented real-time interaction with the GitHub API using fetch and async/await.

  • Processed JSON responses to display user profiles and their public repositories.

  • Used dynamic component rendering with props and conditional display (v-if / v-for).

  • Handled errors gracefully (e.g., user not found via response.ok check).

  • Styled the application using Bootstrap for responsive UI design.
